**SQL Analýza prodejů a ziskovosti (SuperStore)**



**Popis projektu**

Tento projekt simuluje reálný analytický úkol pro vedení společnosti SuperStore. Cílem bylo transformovat surová data o objednávkách do přehledné struktury, která umožňuje identifikovat ziskovost napříč kategoriemi a produkty.


**Cíl analýzy**

- Zjistit celkové tržby a zisky podle kategorií a podkategorií.

- Identifikovat nejprodávanější produkty (podle množství).

- Výpočet procentuální marže pro hodnocení úspěšnosti sortimentu.


**Použité technologie a dovednosti**

- SQL (Keboola / Snowflake): Čištění dat, agregační funkce, seskupování a výpočty metrik.

- Business Intelligence: Interpretace dat pro potřeby managementu.

- Tableu: Tvorba interaktivního dashboardu pro vizualizaci výsledků.


**Klíčová zjištění**

- Kategorie TECHNOLOGY generuje nejvyšší tržby, ale množství prodaných kusů je nižší než u kancelářškých potřeb.

- Z prvního grafu je patrný nárůst zisku v posledním kvartálu roku 2016.
