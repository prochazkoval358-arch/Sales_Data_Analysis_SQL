**SQL Analýza prodejů a ziskovosti (SuperStore)**



**Popis projektu**

Tento projekt se zaměřuje na analýzu obchodních výsledků fiktivní společnosti SuperStore. Cílem bylo zpracovat surová data o objednávkách a poskytnout managementu jasné odpovědi na otázky týkající se výkonnosti jednotlivých produktových kategorií.


**Cíl analýzy**

Zjistit celkové tržby a zisky podle kategorií a podkategorií.

Identifikovat nejprodávanější produkty (podle množství).

Vypočítat průměrnou marži pro jednotlivé segmenty.


**Použité technologie a dovednosti**

SQL (Keboola / Snowflake): Agregace, seskupování a filtrování dat.

Business Intelligence: Interpretace dat pro potřeby managementu.
